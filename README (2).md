## 🧭 Working Principles

This portfolio follows a documented integrity standard:

- **Dual-route validation** — headline figures are recomputed by two independent methods before publication
- **Baseline-first targets** — improvements are measured against declared baselines, never asserted
- **Declared substitutions** — where public datasets substitute for proprietary data, this is stated in the project README
- **AI-assisted, human-owned** — AI tools (Claude) were used as a drafting and review partner; all analysis, code execution, validation, and conclusions are my own work, and I offer a live demonstration of any project on request
- **Operator–Analyst framework** — methods are mapped to CRISP-DM, DIKW, PDCA and lean principles (see `/docs`)
